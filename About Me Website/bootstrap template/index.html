Learning Basic Web Development (HTML & CSS)

Find Out More

What is this workshop about?

This workshop aims to teach FEMMESHacks participants the basic fundamentals of web development. The two main languages used to create a website are HTML and CSS, which is what this workshop will focus on. We have provided this Bootstrap template for you as a jumping off point - feel free to edit as much or as little as you would like!
